# Twitch Clip Todo

Bookmark moments during Twitch streams to create clips later.

## Features

- Record timestamps with keyboard shortcut or floating button
- Works with live streams and VODs
- Side Panel for managing your bookmarks
- Automatic VOD linking after streams end
- One-click navigation to Twitch's clip creator
- No Twitch account or API authentication required

## Install

[Install from Chrome Web Store](https://chromewebstore.google.com/detail/twitch-clip-todo/EXTENSION_ID)
