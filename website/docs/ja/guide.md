# 使い方ガイド

## インストール

1. [Chrome ウェブストア](https://chromewebstore.google.com/detail/twitch-clip-todo/EXTENSION_ID)からインストール
2. 拡張機能をピン留めしてアクセスしやすく
3. Twitch の配信ページへ移動

## 基本的な使い方

### ステップ 1: 瞬間を記録する

ライブ配信または VOD を視聴中に:

**キーボードショートカット:** `Alt+Shift+C` を押す

**または記録ボタンをクリック**

### ステップ 2: メモを追加（任意）

記録後、メモ入力が表示されます:

- 何が起きたかメモを入力
- `Enter` で保存、`Escape` でスキップ

### ステップ 3: 記録を確認

拡張機能アイコンをクリックしてサイドパネルを開く:

- 記録は配信者ごとにグループ化
- 必要に応じて編集・削除

### ステップ 4: VOD を待つ（ライブ配信の場合）

ライブ配信の記録は VOD にリンクする必要があります:

1. 配信終了を待つ
2. 配信者の VOD ページを訪問
3. 記録が自動的に VOD にリンク

### ステップ 5: クリップを作成

1. VOD リンク済みの記録でクリップアイコンをクリック
2. Twitch のクリップ作成ページが正しい位置で開く
3. Twitch のインターフェースでクリップを作成

## フローティングウィジェット

現在の配信者に未処理の記録がある場合:

- **ドラッグ** で画面上の好きな位置に移動
- **クリック** でサイドパネルを開く
- **バッジ** で未処理の記録数を表示

## キーボードショートカット

| 操作       | ショートカット                                                       |
| ---------- | -------------------------------------------------------------------- |
| 瞬間を記録 | `Alt+Shift+C`（chrome://extensions/shortcuts でカスタマイズ可能） |

## Tips

- 記録は多めに - 後で削除できます
- 瞬間が新鮮なうちにメモを追加
- VOD ページを訪問してライブ記録をリンク
