# 機能一覧

## コア機能

### タイムスタンプ記録

- キーボードショートカット記録（`Alt+Shift+C`）
- ワンクリックボタン記録
- ライブ配信と VOD に対応
- オプションのメモ機能

### 記録管理

- ポップアップインターフェース
- 配信者ごとのグループ化
- 記録の編集・削除
- VOD リンク状態の表示
- 完了トラッキング

### クリップ作成

- Twitch クリップ作成ページへワンクリック遷移
- 自動位置計算
- 作成済みマーク

## スマート機能

### 自動 VOD リンク

ライブ配信中に記録した場合:

1. 記録が配信 ID と共に保存される
2. 配信終了後、VOD ページを訪問
3. 拡張機能が自動的にマッチングしてリンク

### フローティングウィジェット

- ドラッグ可能（位置を記憶）
- 待機時に自動非表示
- 未処理の記録数を表示
- クリックでポップアップを開く

### データ管理

- 全データをローカルに保存（Chrome ストレージ）
- 外部サーバーなし
- 60日後に自動クリーンアップ

## プライバシーとセキュリティ

- 簡単なTwitchデバイスコード認証
- 全データがデバイス上に保存
- 全データがデバイス上に保存
- 分析やトラッキングなし
