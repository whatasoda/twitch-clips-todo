# プライバシーポリシー

_最終更新: 2025年1月25日_

## 概要

Twitch Clip Todo はあなたのプライバシーを保護することに取り組んでいます。この拡張機能は、個人データを外部サーバーに収集、送信、保存しません。

## ローカルに保存されるデータ

すべてのデータは Chrome の組み込みストレージ API を使用してデバイスに保存されます:

### 記録データ

- タイムスタンプ（秒単位）
- メモ（オプション）
- 配信者名
- VOD ID（リンク済みの記録）
- 作成タイムスタンプ

### オプションの Twitch データ

Twitch アカウントを接続した場合:

- キャッシュされた配信者情報
- キャッシュされた VOD メタデータ
- 認証トークン（安全に保存）

## 権限

| 権限        | 目的                           |
| ----------- | ------------------------------ |
| `tabs`      | 現在の Twitch ページを検出     |
| `storage`   | 記録をローカルに保存           |
| `activeTab` | Twitch プレーヤーと連携        |
| `alarms`    | 自動クリーンアップをスケジュール |

## サードパーティサービス

- **Twitch API**（オプション）: 認証時の拡張機能に使用
- 分析やトラッキングサービスなし

## データの削除

すべてのデータを削除するには:

1. 拡張機能アイコンを右クリック
2.「Chrome から削除」を選択

または Chrome 設定で拡張機能データをクリア。
