# プライバシーポリシー

_最終更新日: 2026年2月_

## 概要

Twitch Clip Todo はお客様のプライバシーを保護することをお約束します。このポリシーでは、収集するデータとその使用方法について説明します。

## 単一目的

Twitch Clip Todo には明確な単一の目的があります：**Twitch 配信中の気になる瞬間をブックマークし、後でクリップを作成すること**

すべての機能はこのコア機能をサポートしています：

- ライブ配信や VOD 視聴中のワンクリックタイムスタンプ記録
- メモや配信者ごとのグループ化による記録管理
- ライブ配信記録の自動 VOD リンク
- Twitch クリップ作成ページへのワンクリック遷移

この拡張機能は、関連のない機能をバンドルしていません。

## データ収集

**Twitch Clip Todo は、外部サーバーに個人データを収集、送信、保存することはありません。**

すべてのデータは Chrome の組み込みストレージ API（`chrome.storage.local`）を使用して、お客様のデバイスにローカルで保存されます。

## ローカルに保存されるデータ

Twitch Clip Todo は以下のデータをデバイスにローカルで保存します：

### 記録データ

- **タイムスタンプ**: 配信や VOD 中にブックマークした瞬間（秒単位）
- **メモ**: その瞬間を説明するためのオプションのメモ
- **配信者名**: 視聴中の配信者の名前
- **VOD ID**: ライブ配信の記録をアーカイブ VOD にリンクするための識別子
- **作成タイムスタンプ**: 各記録が作成された日時

### オプションの Twitch データ

拡張機能を Twitch アカウントと接続した場合：

- キャッシュされた配信者情報（表示名、プロフィール画像）
- キャッシュされた VOD メタデータ（タイトル、再生時間）
- 認証トークン（Chrome のローカルストレージに保存）

### ユーザー設定

- ウィジェットの位置設定
- 拡張機能の動作設定

## 収集しないデータ

- 個人識別情報（名前、メールアドレス、住所）
- Twitch 以外の閲覧履歴
- 分析データや利用統計
- 広告識別子
- 位置情報
- Twitch 以外のウェブサイトのデータ

## データ保持期間

- 記録はスケジュールされたアラームにより **60日後** に自動的にクリーンアップされます
- 個別の記録やすべてのデータはいつでも手動で削除できます
- キャッシュされた Twitch データは定期的に更新され、不要になった時点で削除されます

## 使用する権限

Twitch Clip Todo は以下の Chrome 権限を必要とします：

| 権限        | 用途                                                                                    |
| ----------- | --------------------------------------------------------------------------------------- |
| `tabs`      | 現在の Twitch ページの URL とタイトルを読み取り、配信者とコンテンツの種類を識別する     |
| `storage`   | ブックマーク記録、ユーザー設定、キャッシュデータをデバイスにローカル保存する            |
| `activeTab` | 現在のタブの Twitch プレーヤーと連携し、正確なタイムスタンプを取得する                  |
| `alarms`    | 60日以上経過した記録の自動クリーンアップをスケジュールする                               |

## サードパーティサービス

### Twitch API（オプション）

Twitch アカウントを接続することを選択した場合、Twitch Clip Todo は以下の目的で Twitch API と通信します：

- ライブ配信記録をリンクするための VOD 情報の取得
- 表示用の配信者プロフィール情報の取得

この通信は、これらの機能を明示的に使用する場合にのみ行われます。Twitch の公式 API 以外のサーバーにデータが送信されることはなく、標準的な Twitch API リクエストのみが行われます。

### 分析やトラッキングなし

Twitch Clip Todo はサードパーティの分析、トラッキング、広告サービスを一切使用しません。

## データ共有

サードパーティとデータを共有することはありません。すべてのデータはお客様のデバイスに留まります。

## データ削除

Twitch Clip Todo のすべてのデータを削除するには：

1. 拡張機能アイコンを右クリック
2.「Chrome から削除」を選択

これにより、記録、設定、キャッシュされた Twitch データを含む、ローカルに保存されたすべてのデータが削除されます。

Chrome 設定で拡張機能データをクリアするか、拡張機能のポップアップから個別の記録を削除することもできます。

## お客様の責任

Twitch Clip Todo はすべてのデータをローカルに保存し、外部に送信することはありませんが、以下についてはお客様の責任となります：

- 不正アクセスからデバイスを保護すること
- ブラウザプロファイルへのアクセス権限を持つ者を管理すること
- ローカルに保存されたデータはデバイスにアクセスできる人なら誰でもアクセス可能であることを理解すること
